## 2️⃣ 생각

사전을 사용하자.

key: 의상의 종류, value: 해당 종류의 의상의 개수

상의 2개 하의 1개라면 가능한 경우의 수는 몇가지일까?

상의 1개 고를때 가능한 경우의 수 : 상의1 상의2 상의x = 3

하의 1개 고를때 가능한 경우의 수 : 하의1 하의x = 2

총 경우의 수 : 3 * 2 - 1

아무것도 안고르는 경우는 빼기때문에 마지막에 -1을 해준다.



## 3️⃣ 풀이

### 🔸성공한 풀이

```python
def solution(clothes):
    ans = 1
    dic = {}
    
    for c in clothes:
        if c[1] in dic:
            dic[c[1]] += 1
        else:
            dic[c[1]] = 1
    
    for val in dic.values():
        ans = ans * (val + 1)
        
    return ans -1
```
