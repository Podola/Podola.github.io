## 3️⃣ 풀이

### 🔸성공한 풀이

```python
def solution(clothes):
    ans = 1
    dic = {}
    
    for c in clothes:
        if c[1] in dic:
            dic[c[1]] += 1
        else:
            dic[c[1]] = 1
    
    for val in dic.values():
        ans = ans * (val + 1)
        
    return ans -1
```
